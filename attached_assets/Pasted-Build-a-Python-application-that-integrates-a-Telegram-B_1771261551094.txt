Build a Python application that integrates a Telegram Bot with a "Co-Branded" Proposal Generator.

**Context:**
I am "Sparktoship", a Solution Architecture firm. I need a bot that generates custom "Pre-Sales" proposals. When I send a client URL to the bot, it creates a co-branded HTML page featuring the Client's Brand AND my Brand (Sparktoship).

**My Brand Assets (Hardcoded):**
* **Name:** Sparktoship
* **Website:** https://sparktoship.com
* **Logo:** (Use a text placeholder "Sparktoship" or a FontAwesome icon for now).

**Tech Stack:**
* Python (Flask, pyTelegramBotAPI, Requests, BeautifulSoup, threading).
* Node.js (for `@mermaid-js/mermaid-cli`).

**File Requirements:**

1.  **`generator.py` (The Logic):**
    * Function `build_proposal(client_url, project_name)`:
        * **Scrape Client:** Fetch `client_url`. Find a logo (favicons, meta tags). Find a primary color (default to Blue/Gray).
        * **Generate Diagram:** Create a Mermaid graph: `Client App -> Sparktoship API Gateway -> Microservices -> Database`.
        * **Compile Image:** Run `npx -p @mermaid-js/mermaid-cli mmdc -i temp.mmd -o static/architecture.png`.
        * **Render HTML:** Pass both `client_data` (logo, color) AND `my_data` (Sparktoship name, URL) to the template.

2.  **`templates/proposal.html` (The Co-Branded Template):**
    * **Header:** A "Dual-Logo" layout using Flexbox.
        * Left: [Client Logo] (Scraped).
        * Center: " X " (Cross icon or symbol).
        * Right: [Sparktoship Logo/Text].
    * **Hero Section:** "Strategic Architecture Proposal for [Client Name]".
    * **Body:** Display the `architecture.png`.
    * **Footer:** "Engineered by Sparktoship Solutions. Visit sparktoship.com".
    * **Style:** Use Tailwind CSS. The Client's primary color should be the accent color (buttons/borders).

3.  **`main.py` (The Bot & Server):**
    * **Telegram Handler:**
        * Listen for text like: "/pitch [url] [project_name]".
        * Reply "Analyzing [URL] for Sparktoship proposal..."
        * Run generation.
        * Reply with the link and the diagram image.
    * **Flask Server:** Serve the generated page at `/proposal`.

**Output:**
Write the complete code for `main.py`, `generator.py`, `templates/proposal.html` and `requirements.txt`. Ensure the co-branding is visually distinct in the HTML.
